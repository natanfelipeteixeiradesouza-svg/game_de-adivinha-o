## 📋 Sobre o Projeto

Este é um jogo de adivinhação desenvolvido como parte da avaliação da disciplina **Projeto de Games** do curso **2º Áudio e Vídeo** do **IFG Câmpus Cidade de Goiás - 2025**.

O objetivo é simples: adivinhe o número secreto gerado aleatoriamente pelo sistema! A cada nova partida, um número é sorteado e o jogador deve usar suas tentativas de forma estratégica para descobrir qual é o número misterioso.

## 🎮 Como Jogar

1. Ao abrir a página web, um número secreto é automaticamente sorteado
2. Digite seu palpite no campo indicado
3. Clique em "Tentar" ou pressione Enter
4. Receba dicas se seu número é maior ou menor que o número secreto
5. Continue tentando até acertar ou esgotar suas chances
6. Quando o jogo terminar, você pode iniciar uma nova partida

## ✨ Funcionalidades

- 🎯 Geração aleatória de números secretos
- 💡 Sistema de dicas (maior/menor)
- 📊 Contador de tentativas
- 🏆 Feedback visual para vitória/derrota
- 📱 Interface responsiva
- 🔄 Opção de reiniciar o jogo

## 🚀 Tecnologias Utilizadas
